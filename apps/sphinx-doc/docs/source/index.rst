Docker Sphinx Documentation
===========================

Overview
--------

Welcome to Docker Sphinx's documentation!

This is a Sphinx documentation project that demonstrates how to use Docker to build and run a Sphinx documentation.

Main goals of this project
--------------------------

- Easy and fast setup of a Sphinx documentation project
- Have build in documentation how to use this Docker Sphinx image
- Generate HTML documentation using Docker image
